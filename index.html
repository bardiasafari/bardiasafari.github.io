What is Photoshop?

If you are reading this webpage, you have probably already heard of the application Photoshop, but want to know more about it. Adobe Photoshop is a very useful and complex program that is used by many professional artists and designers. Photoshop can be used for photography, logo making, image editing, photo touch-ups, and more!

Price

Photoshop is very expensive, but many users say that is is worth trying out and buying. Photoshop, along with most other Adobe Apps, comes in a program called the Creative Cloud, also referred to as CC. Users have the option to try out a seven day free trial, and purchase the app if they like it. Many users that have used Photoshop have learned quickly and loved it, but if you can't afford it, you can always try free or cheaper Photoshop alternatives, like Paint.NET.

Image Formats

Photoshop can save edited images in a ton of formats, likes PNG, JPG, and more. You can also save a Photoshop file as a .PSD file, which lets you edit it later if you want. PSD files are very commonly used in graphic design files. Photoshop also uses a useful system, commonly known as layers. Layers help users organize their graphic design and images by separating the parts. For example, the background can be on one layer while an image of a character is on another. So when someone wants to erase the character, they can use that layer, which will help so they don't erase the background too.

To create a new document:

  1. Locate the systems bar
  2. Select "File" > "New.."
  3. Name your document
  4. Select a preset or input custom dimensions and color types

To open a document:

  1. locate the systems bar
  2. Select "File">"Open"(Command+O on Mac, Ctrl+O on Windows)
  3. Find the file you are trying to open
  4. Select "Open"
